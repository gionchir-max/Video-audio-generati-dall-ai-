## Note operative meta.ai

- **PRIMA di inviare**: cliccare bottone "Crea video" sulla home
- **Una chat ogni 3 clip** (per evitare freeze del bottone Invia)
- **Ogni prompt INIZIA con `5-second cinematic video clip:`** + `vivid/vibrant saturated/punchy colors`
- **Verbo movimento esplicito** in ogni prompt
- **Linguaggio neutro** (no "wife/baby" insieme; "the moustachioed artisan" invece di nomi specifici)
- **Salva i video** scaricati come `clips/01.mp4` … `clips/36.mp4`
- Se meta.ai risponde con "Oops" → skippa e continua

## Quando hai tutti i clip

```bash
npm run video nutella
# → videos/nutella/out/final.mp4
```
